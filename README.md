# DSCI510_Final_Project
Overview:
This project scrapes book data from “Books to Scrape”, cleans the dataset, runs analysis, and generates basic visualizations.

1. Install Requirements
   pip install -r requirements.txt
2. Download the Raw Data
   python src/get_data.py
3. Clean the Data
   python src/clean_data.py
4. Run Analysis
   python src/run_analysis.py
5. Generate Visualizations
   python src/visualize_results.py
   This step produces: price_distribution.png, rating_distribution.png, price_vs_rating.png, top10_expensive_books.png

Project Structure
data/raw/                Raw scraped data
data/processed/          Cleaned dataset
results/                 All generated plots + final report
src/                     All Python scripts
